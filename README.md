# AWF Marketplace

Claude Code plugin marketplace for AWF [CLI](https://github.com/awf-project/cli).

## Installation

```bash
/plugin marketplace add awf-project/awf-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| `awf` | AWF - skills, agents, and commands |
